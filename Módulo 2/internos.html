Dispositivos Periféricos Internos

Os dispositivos periféricos internos comuns encontrados em um computador incluem uma unidade de disco óptico, uma placa de vídeo e um disco rígido. Nesses exemplos, a unidade de disco é uma instância de um dispositivo que é um dispositivo de entrada e de saída. Ele não pode apenas ser usado pelo computador para ler as informações armazenadas no disco (por exemplo, software, música, filmes), mas também para exportar dados do computador para o disco (como ao gravar DVDs).

Quais são e para que servem?

Placa-mãe- Acopla todos os componentes de um computador, ou seja, é onde todos os equipamentos se encaixam. É uma placa de circuítos composta de caminhos de dados (barramentos) e lacunas para encaixar os equipamentos (slots).

Processador: É o "cérebro" do computador. Ele também é chamado de micro processador ou CPU e é responsável por processar as informações do PC. Os grandes fabricantes de processador são: Intel e AMD. A frequencia "velocidade" do processador chama-se: Clock.

Memória: a função da memória é armazenar dados. Existem diversos tipos de memórias: memórias permanentes e virtuais, cada uma com função definida: Principal (RAM e ROM) Auxiliar (Virtual e Cache) Secundária (HD, floppy, CD/DVD-ROM, etc)

Memória RAM - (memória de acesso aleatório) é a mais importante. Só funciona com o computador ligado, por isso, é chamada de volátil. A CPU é que mais utiliza esse tipo de memória.

Memória ROM - (memória somente para leitura) armazena dados importantes do fabricante do equipamento e nao podem ser utilizadas pelo usuário nela todos os dados básicos para o PC funcionar. Ao conjunto formado pelas memorias RAM e ROM dá-se o nome de Memória principal

Memória Cache - Encontra-se no processador e trabalhaem sincronia com a RAM, porém ela armazena dados mais rápido, é um tipo de RAM estática: é uma SRAM. Sua função é armazenar os dados mais recentes requisitados pela RAM principal.

Fonte de Alimentação - É o módulo que fornece energia elétrica ao micro. As fontes de alimentação utilizadas hoje em dia são do tipo chaveada (aumenta o rendimento e torna a fonte mais compacta).

HD-Disco Rígido - É o locar onde se instala o sistema operacional (Windows, Linux), e onde armazenamos os nossos arquivos(word, excel, pastas, fotos, vídeos, etc..). Nele cria-se uma memória virtual quando necessário. Pode também ser chamado de Winchester.

Unidade de CD/DVD - Faz a leitura ou/e gravação de mídias de CD e DVD. A capacidade de armazenamento do CD foi originalmente definida em 650MB, mas hoje é 700MB. Já o DVD uma capacidade de 4,7GB, ou seja, quase 7 vezes a capacidade de CD. Os tipos mais comuns de CDs e DVDS são: CD/DVD-ROM: disco que ja vem gravado com informações de fábrica. É um CD/DVD apenas para leitura. CD/DVD-R: é o CD/DVD virgem. Só pode ser gravado uma vez e significa CD/DVD gravável CD/DVD-RW: é um CD/DVD regravável. Pode ser gravado várias vezes.

Placa de rede - É um dispositivode entrada e saída de dados e permite a comunicação através da estrutura de cabeamentos. É também conhecida como: Adaptador de rede, NIC (placa de interface de rede) ou placa ethernet (por causa da arquitetura usada).

Placa de som - É um dispositivo de entrada e saída de informações que permite transformar as informações digitais dos programas e jogos em som stereo através das caixas de som. Sem ela não seria possível ouvir os sons de jogos e programas, nem tão pouco ouvir músicas, ouvir o áudio de filmes, etc.

Placa de Vídeo: É um dispositivo de saída de dados que permite que os dados apareçam no monitor de vídeo do computador. Todos os dados que saem da CPU em direção ao monitor passam pela Placa de Vídeo, que converte os sinais elétricos digitais em sinais RGB ( cores primárias ). Existem também as placas de vídeo 3D ( conhecidas como: Placas Aceleradoras ), elas trazem um processador próprio que realiza cálculos de ambientes tridimensionais. Todas as placas de vídeo possuem memória, a chamada Memória de Vídeo, que nada mais é que a memória DRAM utilizada apenas para armazenar os dados de imagem ( pixels e suas cores). Quanto mais memória de vídeo maior a sua resolução e mais cores a imagem do computador pode apresentar.
